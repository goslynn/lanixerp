Panel de Control de Avisos

Este correspode al aviso fijo que se despliega en cada Aplicacion, este aviso depende de la version de la herramienta que este utilizando. Esta noticias solo es visible en Panel de Control de Avisos.

Tiempo: 10000ms

Secciones

Panel Izquierdo : Lista de Noticias Publicadas

Panel Central : Visor de Noticias

Seccion Inferior: Control

USO de PCA

Seleccionar el Target

Ingresar una URL en linea, accesible desde internet

Selecciona el tiempo que se mantiene en pantalla

Usa el boton ( Ir ) para visualizar la noticia

Usa el boton ( + ) para publicar la noticia

Usa el boton ( > ) para simular la ronda de noticias.

Usa el boton ( R ) para recargar manualmente lista de noticias

Para eliminar una noticia, selecciona la noticia del panel a la izquierda y presione ( - )
